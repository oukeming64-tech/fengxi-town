# Reusable Skills From Fengxi Town

These Codex skills capture reusable methods extracted from the Fengxi Town project. They are intentionally general: the town's private source material stays out of the public skill text, while the repeatable architecture and publication workflows are preserved.

- `ai-sim-fact-ledger`: keep AI simulations grounded in deterministic local facts.
- `world-rules-normalizer`: turn narrative world rules into traceable runtime modules.
- `resident-cognition-loop`: design audited multi-agent resident behavior loops.
- `llm-contract-validation-harness`: validate structured model output with guards, sharding, retries, and fixtures.
